# Sales Data Analysis Project

## About This Project
This project was created while learning Python Data Analysis.

The goal was to understand how real sales data can be analyzed to generate useful business insights.


## Project Overview
In this project, I analyzed sales data to understand:

- Product performance  
- Total revenue  
- Monthly sales trends  


## Objectives
- Understand how data analysis works in real-world business problems  
- Learn how to clean and process CSV data  
- Generate insights using Python libraries  


## Tools Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  


## What I Did
- Loaded and cleaned the sales dataset  
- Created a new column for total sales  
- Grouped data by product and category  
- Calculated total revenue  
- Analyzed monthly sales trends  


## Key Insights
- Some products generated high revenue even with low quantity sold  
- Electronics category performed better than stationery  
- Sales showed variation across different months  


## Visualization
- Monthly sales trends were analyzed using a line chart  


## Conclusion
This project helped me understand the basics of data analysis, data cleaning, and visualization using Python. It improved my ability to extract meaningful insights from raw data.
